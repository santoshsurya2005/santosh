Stack & Queue

Stacks and Queues are fundamental data structures used to solve various computational problems. They provide different ways to store and process data.

1. Stack

A Stack is a linear data structure that follows the Last In, First Out (LIFO) principle. The last element added to the stack is the first to be removed.

  • Push: Add an element to the top of the stack.
  • Pop: Remove the top element from the stack.
  • Peek: View the top element without removing it.
#include <stdio.h>
#define MAX 100

int stack[MAX], top = -1;

void push(int value) {
    if (top == MAX - 1) {
        printf("Stack Overflow\n");
    } else {
        stack[++top] = value;
    }
}

int pop() {
    if (top == -1) {
        printf("Stack Underflow\n");
        return -1;
    } else {
        return stack[top--];
    }
}

int peek() {
    if (top == -1) {
        printf("Stack is empty\n");
        return -1;
    } else {
        return stack[top];
    }
}

int main() {
    push(10);
    push(20);
    printf("Top element is %d\n", peek());
    printf("Popped element is %d\n", pop());
    return 0;
}

2. Queue

A Queue is a linear data structure that follows the First In, First Out (FIFO) principle. The first element added to the queue is the first to be removed.

  • Enqueue: Add an element to the rear of the queue.
  • Dequeue: Remove an element from the front of the queue.
  • Front: View the front element without removing it.
#include <stdio.h>
#define MAX 100

int queue[MAX], front = -1, rear = -1;

void enqueue(int value) {
    if (rear == MAX - 1) {
        printf("Queue Overflow\n");
    } else {
        if (front == -1) front = 0;
        queue[++rear] = value;
    }
}

int dequeue() {
    if (front == -1 || front > rear) {
        printf("Queue Underflow\n");
        return -1;
    } else {
        return queue[front++];
    }
}

int peek() {
    if (front == -1 || front > rear) {
        printf("Queue is empty\n");
        return -1;
    } else {
        return queue[front];
    }
}

int main() {
    enqueue(10);
    enqueue(20);
    printf("Front element is %d\n", peek());
    printf("Dequeued element is %d\n", dequeue());
    return 0;
}
